## How to run locally


First of all, clone this repository to your Device,by using the following command:

```bash
git clone https://github.com/0xmuon/ASPCA-website
```

Then inside the ASPCA-website folder,using Terminal,run the following command to install dependencies:
```bash
npm install
```

Finally run the following in Terminal:
```bash
npm run dev
```

You can now access the localhost.
